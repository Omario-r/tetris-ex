# VISION.md — Explosive Template Tetris

## Одна фраза

Мобильная головоломка, где тетрис-механика служит инструментом точной сборки:
падающие фигуры укладываются в заданный шаблон, а взрывной режим позволяет
хирургически убрать всё лишнее.

## Проблема

Классический тетрис — рефлекторная аркада без нарративного вознаграждения.
Казуальные паззлы (match-3 и др.) лишены физики укладки. Нет жанра, который
давал бы удовлетворение от «красивой сборки» детали при тактическом управлении
падающими блоками.

## Решение

Каждый уровень задаёт шаблон (Target). Игрок укладывает тетромино так, чтобы
точно заполнить шаблон, и при необходимости использует взрывной режим, чтобы
точечно удалить мусор. Победа — не «очисти линию», а «собери деталь идеально».

## Целевая аудитория

Казуальные игроки 18–35 лет, ценящие паззлы с ощущением прогресса.
Коллекционирование объектов из фрагментов создаёт мета-петлю удовлетворения.

## Ключевая гипотеза

Механика взрывного режима (точечное удаление мусора) делает паззл управляемым
и тактическим, не убирая напряжение от укладки фигур.

## Формат сессии

Короткий уровень 2–4 минуты + глубокая мета-петля.
Игрок открывает приложение 4–6 раз в день, каждый раз проходит 1–2 уровня.
Возврат мотивирован мета-прогрессией (фрагмент → объект → большая картина).

Структура одной сессии:
ObjectPreview → Target 1 → пауза → Target 2 → пауза → Target 3 → пауза → Target 4 → ObjectComplete

Сессия не прерывается между фрагментами одного Объекта — игрок в потоке.
После каждого фрагмента: короткая анимация (фрагмент встаёт на место) +
кнопки «Продолжить» и «Сохранить и выйти». Прогресс сохраняется, вернуться
можно в любой момент.

## ObjectPreview — якорь мотивации

Перед началом Объекта игрок видит его целиком с пунктирными линиями,
разделяющими на Target-фрагменты. Уже собранные фрагменты закрашены,
текущий выделен. Игрок знает куда идёт до первого хода.

## Система очков (гибрид)

Базовый счёт:
- +10 за фиксацию фигуры в зону S
- +50 за детонацию взрывной фигуры
- +200 за победу на уровне

Множители:
- x2 если фигура легла целиком в S без единой клетки мимо
- +100 за чистый взрыв (убрал только мусор, S не задета)
- Цепочка точных фиксаций без взрыва: x1.5 → x2 → x3

Штрафы (мягкие, не блокируют прохождение):
- -20 за каждую грязную клетку Halo в момент победы
- Взрыв над пустой зоной — потерял инструмент, штрафа нет

Итог уровня — звёзды 1–3:
- 1 звезда — уровень пройден (любой счёт)
- 2 звезды — собрал аккуратно
- 3 звезды — мастерская сборка, достижимо с 3–4 попытки

## Pure Path — чистая сборка

Некоторые Target специально проектируются так, что их можно собрать
без единого взрыва: тетромино покрывают S как пазл, не задевая Halo.
Условие: площадь S кратна 4, форма тайлится стандартными тетромино.

Маркировка:
- Метка ✦ на карте уровней и превью Target
- В начале сессии одноразовое сообщение: «Этот шаблон можно собрать без взрывов»
- В итогах: строка «✦ Чистая сборка — бонус +300» (серая если не достигнута)

Два типа уровней:
- Обычный Target (без метки) — взрывной путь, Halo допустим
- Чистый Target (✦) — можно собрать как пазл без взрывов

В LevelTemplate добавляется поле hasPurePath: bool.

Чистые Target управляют кривой сложности: обычные уровни — обучение и отдых,
чистые — вызов и мастерство.

## Скоуп

В прототипе (MVP):
- 1 уровень — крест 6x6
- Normal + Explosive режим
- Halo-индикатор (8-соседство)
- Линейная мета-прогрессия, 16 уровней

За скоупом:
- Генерация уровней
- Мультиплеер
- Магазин / IAP
- Большая картина (4 объекта)

## Нарративный слой

Игрок собирает объекты (первый — «Домик») фрагмент за фрагментом.
После каждого уровня фрагмент анимированно встаёт на своё место в объекте.
После 4 фрагментов объект собран целиком. Простой прогресс без сюжета,
понятный с первого взгляда.

## Метрики успеха прототипа

- Игрок понимает механику без туториала (первые 2 уровня).
- Игрок применяет взрывной режим осознанно, а не спамит.
- Уровень проходится за разумное число ходов (нет ощущения безысходности).
- Игрок доходит до конца первого Объекта без принудительного выхода.
