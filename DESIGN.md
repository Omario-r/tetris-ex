# DESIGN.md — Explosive Template Tetris

## Core loop

Каждый уровень задаёт шаблон (Target).
Игрок укладывает падающие тетромино так, чтобы заполнить целевую форму.
Если вокруг шаблона накопился мусор, игрок может использовать взрывной режим
для точечного удаления лишних клеток.

Победа — шаблон собран.
Фокус игры — не очистка линий, а аккуратная и точная сборка.

## Сессия

Короткий уровень длится 2–4 минуты.
Типичная сессия: 1–2 уровня или один модуль из нескольких фрагментов.

Структура модуля:
ObjectPreview → Фрагмент 1 → пауза → Фрагмент 2 → пауза → Фрагмент 3 → ObjectComplete

Сессия не прерывается между фрагментами одного модуля.
После каждого фрагмента игрок видит короткую анимацию и кнопки:
- Продолжить
- Сохранить и выйти

## ObjectPreview

Перед началом модуля игрок видит объект целиком с разделением на фрагменты.
Уже собранные части закрашены, текущая часть выделена.
Экран нужен как якорь мотивации: игрок понимает, к чему идёт.

## Мета-прогрессия

Тема: космос. Игрок собирает детали космических кораблей.

| Термин | Состоит из | Уровней |
|---|---|---|
| Фрагмент | 1 Target | 1 |
| Объект (модуль корабля) | 3 фрагмента | 3 |
| Большая картина (корабль) | 5 модулей | 15 |
| Коллекция (флот) | 10 кораблей | 150 |

Галерея содержит 10 силуэтов кораблей, которые постепенно заполняются.
Финальная цель — собрать весь флот.

## Система очков

Базовый счёт:
- +10 за фиксацию фигуры в зону S
- +50 за детонацию взрывной фигуры
- +200 за победу на уровне

Множители:
- x2 если фигура легла целиком в S без клеток мимо
- +100 за чистый взрыв, если убран только мусор и S не задета
- Цепочка точных фиксаций без взрыва: x1.5 → x2 → x3

Штрафы:
- -20 за каждую грязную клетку Halo в момент победы
- Взрыв над пустой зоной расходует инструмент без штрафа

Итог уровня:
- 1 звезда — уровень пройден
- 2 звезды — аккуратная сборка
- 3 звезды — мастерская сборка

## Combo meter

Combo meter растёт за точные фиксации и чистые взрывы.
Потолок — x3.
При выходе из сессии сбрасывается до x1.

## Взрывы

- 5 базовых взрывов на уровень
- +2 бонусных взрыва за ежедневную награду
- Неиспользованные базовые взрывы переходят в следующий фрагмент
- Бонусные взрывы между фрагментами не переносятся
- При выходе из сессии остаток сгорает
- armExplosive() — no-op, если взрывов не осталось

## Pure Path

Некоторые шаблоны проектируются так, чтобы их можно было собрать без взрывов.
Условие: площадь S кратна 4, а форма тайлится стандартными тетромино.

Маркировка:
- ✦ на карте уровней
- ✦ в превью Target
- одноразовое сообщение в начале сессии
- бонус +300 в итогах за чистую сборку

Типы уровней:
- Обычный Target — взрывной путь, Halo допустим
- Чистый Target (✦) — шаблон собирается без взрывов

В `LevelTemplate` добавляется поле:
- `hasPurePath: bool`

## Retention

### Уровень 1 — ежедневный возврат

Ежедневная награда:
- за первый пройденный уровень дня игрок получает +2 бонусных взрыва

Объект дня:
- 2 фрагмента
- непрерывная сессия
- seed по дате
- переиграть нельзя

### Уровень 2 — удержание в сессии

- Combo meter
- перенос базовых взрывов между фрагментами
- сброс прогресса удержания при выходе из сессии

### Уровень 3 — долгосрочно

- коллекция флота
- достижения
- недельные события

## Достижения

Награда за достижения — только косметика.
Ориентир: много достижений, точное число определяется после прототипа.

Категория «Мастерство»:
- ✦ Чистая сборка — пройти Pure Path без взрывов
- 💥 Пацифист — пройти уровень с 0 использованных взрывов

Категория «Прогресс»:
- 🔧 Первый модуль — собрать первый объект
- 🚀 Первый корабль — собрать первый корабль
- 🌌 Флот собран — завершить коллекцию

Архитектурно:
- достижения = список `AchievementId` в профиле игрока
- триггеры навешиваются на события движка

## MVP

В прототипе:
- 1 уровень — крест 6x6
- Normal + Explosive режим
- Halo-индикатор (8-соседство)
- линейная мета-прогрессия
- 16 уровней

За пределами MVP:
- полная система очков и звёзд
- Pure Path уровни
- объект дня
- ежедневная награда
- Combo meter
- перенос взрывов
- достижения
- галерея флота
- монетизация

## Open questions

Эти темы требуют отдельного разбора перед выходом за пределы MVP.

- Награда за объект дня — механика готова, награда не определена
- Стрик — отложен, вернуться после прототипа
- Категория достижений «Открытия» — отложена, вернуться после прототипа
- Еженедельный челлендж — не разобрали
- Монетизация — предварительно: взрывы как расходник + подписка

## Монетизация

Предварительный принцип:
- взрывы могут продаваться как расходник
- возможна подписка
- Pure Path всегда должен быть достижим без покупок
